筛选命令

Wireshark筛选命令

掌握Wireshark常用筛选语法，快速定位可疑流量

基础筛选语法

IP地址筛选

IPv4
ip.addr == 192.168.1.100

筛选与该IP相关的所有流量（源或目的）

ip.src == 192.168.1.100

筛选源IP为指定地址的流量

ip.dst == 192.168.1.100

筛选目的IP为指定地址的流量

ip.src == 192.168.1.0/24

筛选源IP在指定网段的流量（CIDR表示法）

!(ip.addr == 192.168.1.100)

排除与该IP相关的流量

端口筛选

tcp.port == 80

筛选80端口的TCP流量（源或目的）

tcp.srcport == 80

筛选源端口为80的TCP流量

tcp.dstport == 443

筛选目的端口为443的TCP流量

udp.port == 53

筛选53端口的UDP流量（DNS常用）

tcp.port >= 1 and tcp.port <= 1024

筛选1-1024端口范围的TCP流量

协议筛选

tcp

筛选所有TCP协议流量

udp

筛选所有UDP协议流量

icmp

筛选ICMP流量（ping命令使用）

arp

筛选ARP协议流量

dns

筛选DNS协议流量

协议筛选

HTTP协议筛选

http

筛选所有HTTP流量

http.request.method == "GET"

筛选GET请求

http.request.method == "POST"

筛选POST请求

http.request.uri contains "/admin"

筛选URI包含/admin的请求

http.host == "example.com"

筛选指定Host的请求

http.user_agent contains "sqlmap"

筛选包含sqlmap的User-Agent

http.response.code == 200

筛选HTTP 200响应

http.response.code >= 400

筛选4xx和5xx错误响应

HTTPS/TLS筛选

tls

筛选所有TLS流量

tls.handshake.type == 1

筛选TLS Client Hello（握手请求）

tls.handshake.type == 2

筛选TLS Server Hello（握手响应）

tls.handshake.extensions_server_name == "evil.com"

筛选SNI为指定域名的TLS流量

ssl.handshake.extensions_server_name contains "bank"

筛选SNI包含bank的流量

DNS筛选

dns

筛选所有DNS流量

dns.qry.name == "example.com"

筛选指定域名的DNS查询

dns.qry.name contains "evil"

筛选域名包含evil的DNS查询

dns.qry.type == 16

筛选TXT记录查询（DNS隧道常用）

dns.resp.len > 200

筛选响应长度异常的DNS流量

dns.flags.response == 0

筛选DNS请求（response=0表示请求）

dns.flags.response == 1

筛选DNS响应（response=1表示响应）

Web攻击筛选

SQL注入

http contains "SELECT"

筛选包含SELECT关键字的HTTP流量

http contains "'"

筛选包含单引号的HTTP流量（注入探测特征）

http contains "UNION"

筛选包含UNION的HTTP流量（UNION注入特征）

http.request.uri contains "id=" and http.request.uri contains "'"

筛选URL参数id=且包含单引号的请求

http contains "0x7e" or http contains "0x73"

筛选16进制编码特征

XSS攻击

http contains "<script"

筛选包含script标签的流量

http contains "onerror="

筛选包含onerror事件的流量

http contains "javascript:"

筛选包含JavaScript协议的流量

http contains "alert("

筛选包含alert弹窗函数的流量

http contains "eval("

筛选包含eval函数的流量

路径穿越与文件包含

http.request.uri contains "../"

筛选包含../的URL（路径穿越特征）

http.request.uri contains "..%2F"

筛选URL编码的路径穿越

http.request.uri contains "/etc/passwd"

筛选尝试读取passwd文件的请求

http.request.uri contains "file://"

筛选file协议访问

http.request.uri contains "php://"

筛选PHP协议包装器

WebShell特征

http contains "eval"

筛选包含eval函数的流量

http contains "base64_decode"

筛选包含base64解码的流量

http contains "system("

筛选包含系统命令执行的流量

http contains "exec("

筛选包含命令执行的流量

http.request.uri contains ".php?q="

筛选疑似菜刀WebShell的查询参数

http contains "acw_sc__v2"

筛选蚁剑WebShell特征Cookie

DNS隧道检测

DNS隧道特征筛选

dns.qry.name.len > 50

筛选子域名长度异常的DNS查询（隧道特征）

dns.resp.len > 300

筛选响应数据量大的DNS流量

dns.qry.type == 16

筛选TXT记录查询

dns.qry.type == 15

筛选MX记录查询

dns.qry.type == 2

筛选NS记录查询

DGA域名检测

dns.qry.name matches "[a-z]{30,}\.(com|net|org)"

筛选长随机字母组成的疑似DGA域名

dns.qry.name matches ".*[0-9]{5,}.*"

筛选包含多个连续数字的域名

dns.qry.name contains "farf"

筛选Conficker DGA特征域名

dns.qry.name contains "pos"

筛选Zeus DGA特征域名

加密流量分析

TLS指纹筛选

tls.handshake.ciphersuite == 0x002F

筛选指定密码套件（TLS_RSA_WITH_AES_128_CBC_SHA）

tls.handshake.extensions_server_name == ""

筛选没有SNI的TLS请求（可疑）

tls.record.version == 0x0303

筛选TLS 1.2版本

ssl.handshake.session_ticket.length > 0

筛选使用会话票证的TLS流量

JA3指纹筛选

ssl.handshake.type == 1 and tls.handshake.extensions_server_name contains "cdnx"

筛选常见CDN域名的TLS流量

说明

JA3指纹需要通过Lua插件或Bro/Zeek提取，Wireshark本身不直接支持JA3筛选

高级筛选技巧

逻辑运算符组合

ip.addr == 192.168.1.100 and tcp.port == 80

同时满足两个条件（AND）

ip.addr == 192.168.1.100 or ip.addr == 192.168.1.200

满足任一条件（OR）

http and (ip.addr == 192.168.1.100 or ip.addr == 192.168.1.200)

使用括号改变优先级

!(ip.addr == 192.168.1.100)

取反（NOT）

比较运算符

ip.len > 1400

筛选IP包长度大于1400的流量

tcp.window_size < 1000

筛选窗口大小小于1000的TCP流量

frame.len >= 1000 and frame.len <= 2000

筛选指定长度范围的包

tcp.analysis.ack_rtt > 0.5

筛选ACK延迟大于0.5秒的连接

字符串操作

http.request.uri matches "^/api/.*$"

使用正则表达式匹配URI

ip.addr matches "192\.168\..*\..*"

使用正则匹配IP地址

http.request.header contains "X-Forwarded-For"

筛选包含特定请求头的HTTP流量

tcp.payload contains "cmd"

筛选TCP载荷包含cmd的流量

TCP标志位筛选

tcp.flags.syn == 1

筛选SYN包（连接建立请求）

tcp.flags.ack == 1

筛选ACK包

tcp.flags.fin == 1

筛选FIN包（连接结束）

tcp.flags.reset == 1

筛选RST包（强制断开）

tcp.flags.syn == 1 and tcp.flags.ack == 0

筛选只有SYN标志的包（端口扫描特征）

tcp.analysis.retransmission

筛选TCP重传包

时间与统计筛选

frame.time_relative > 60

筛选相对时间大于60秒的包

tcp.stream eq 0

筛选第一个TCP会话的所有包

tcp.analysis.bytes_in_flight > 100000

筛选飞行中数据量大于100KB的TCP流

应用场景汇总

场景1: 发现内网主机被植入后门

筛选策略
# 1. 筛选内网主机的异常外连
ip.src == 192.168.1.100 and !(ip.dst == 192.168.1.0/24) and tcp.dstport > 1024

# 2. 筛选非工作时间的外连行为
ip.src == 192.168.1.100 and tcp and frame.time >= "22:00:00"

# 3. 筛选异常长连接的TCP流量
ip.src == 192.168.1.100 and tcp and tcp.analysis.duration > 3600

关注点：外部IP、非标准端口、长时间持续连接、异常时间段

场景2: 检测Web应用攻击

筛选策略
# 1. 筛选目录扫描行为（短时间内大量404）
http.response.code == 404 and frame.time_delta < 0.1

# 2. 筛选SQL注入尝试
http.request.uri contains "id=" and (http contains "'" or http contains "UNION")

# 3. 筛选敏感文件访问
http.request.uri contains "/admin" or http.request.uri contains ".php.bak"

关注点：404比例、请求频率、异常参数、敏感路径

场景3: 发现横向移动

筛选策略
# 1. 筛选内网大规模扫描
ip.src == 192.168.1.100 and tcp.flags.syn == 1 and tcp.flags.ack == 0 and ip.dst == 192.168.1.0/24

# 2. 筛选RDP暴力破解
tcp.dstport == 3389 and tcp.analysis.retransmission

# 3. 筛选SMB协议横向移动
smb or smb2 or (tcp.port == 445 and tcp.payload contains " Negotiate")

关注点：内网广播域扫描、远程服务暴力破解、文件共享协议

场景4: 检测数据外传

筛选策略
# 1. 筛选大体积HTTP上传
http.request.method == "POST" and http.content_length > 10000000

# 2. 筛选DNS隧道（大量TXT查询）
dns.qry.type == 16 and frame.time_delta < 1

# 3. 筛选可疑的HTTPS外传
ip.src == internal_ip and tls and ip.dst != known_safe_ips

关注点：大文件上传、DNS异常行为、未知IP的TLS连接

场景5: 分析ICMP隧道

筛选策略
# 1. 筛选大载荷ICMP包
icmp and data.len > 64

# 2. 筛选非常规ICMP类型
(icmp.type == 3 or icmp.type == 11) and ip.dst == internal_ip

# 3. 筛选可疑ICMP流量
icmp contains "cmd" or icmp contains "powershell"

关注点：ICMP载荷大小、异常ICMP类型、数据隐藏

场景6: 发现伪造木马的通信行为

筛选策略
# 1. 筛选心跳机制（固定间隔的请求）
ip.addr == suspicious_ip and tcp and tcp.analysis.ack_rtt approx 30000

# 2. 筛选可疑的TLS Client Hello
tls.handshake.extensions_server_name in {evil.com evil.net}

# 3. 筛选已知恶意软件通信特征
ssl.handshake.session_id.length > 32

关注点：固定心跳间隔、恶意域名TLS连接、会话ID异常
