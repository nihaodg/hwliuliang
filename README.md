# HW流量分析知识库

专为护网行动（HW）设计的流量分析知识库，帮助安全工程师快速识别和分析网络攻击流量。

## 功能特性

- **流量特征原理** - 掌握常见网络协议的攻击流量特征
- **分析方法** - 系统化的流量分析方法和框架
- **Wireshark筛选命令** - 快速定位可疑流量

## 内容模块

### 协议特征
- HTTP/HTTPS流量特征
- DNS协议流量分析
- TCP/UDP流量分析

### Web攻击类型
- SQL注入
- XSS跨站脚本
- SSRF服务端请求伪造
- 命令注入/RCE
- 文件上传攻击
- XXE XML外部实体注入

### WebShell工具流量
- 中国蚁剑 (AntSword)
- 冰蝎 (Behinder)
- 哥斯拉 (Godzilla)
- 菜刀 (China Chopper)

### 渗透工具指纹
- Nessus / AWVS / OpenVAS
- SQLMap / Nmap
- Hydra / Medusa
- Metasploit / PsExec

### 高级分析
- 横向移动流量检测
- 隧道流量识别
- 加密流量深度分析 (DoH/JA3/SSH)
- 典型攻击链分析

## 技术栈

- Tailwind CSS (CDN)
- Lucide Icons
- 纯HTML/CSS/JS，无依赖

## 使用方法

直接打开 `index.html` 即可浏览，支持深色/浅色主题切换。
